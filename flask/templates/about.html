# * disclaimer

+ candle volume cannot accurately determine dealer directional open interest (DDOI), thus one can argue the derived gex is pretty much garbage. but hey, it's free garbage!
+ volatilty and gamma only updated once per minute. dont feel like computing it by the second yet... given above bullet point.

# order imbalance plots

+ call and put order imbalance are sum of 0DTE SPXW candle bid and ask volume every 5min, more selling (red), more buying (green), bins: {{ order_imbalance_bin_str }}.

# gex plot

+ gex is a guesstimate using 0DTE SPXW order imbalance from only the current day (akin @gexbot Classic, or @unusual_whales spot gex volume - not as accurate as @optionsdepth, @spotgamma Trace, @gexbot State)
+ horizontal dashed lines: spot: white, expected moves: gray, major positive gex strike: green, major negative gex strike: red.

# convexity plot

+ 0DTE SPXW order imbalance of the entire day multiplied by current gamma, no put or call distinction. see gexbot document for more info.
+ horizontal dashed lines: major positive convexity: cyan, major negative convexity: purple.
