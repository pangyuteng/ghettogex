# * disclaimer

+ candle volume cannot accurately determine dealer directional open interest (DDOI), thus one can argue the derived gex is pretty much garbage. but hey, it's free garbage!
+ volatilty and greeks for SPX computed with vollib, with ~1sec lag. Greeks for NDX is from data vendor updated once per minute.

# order imbalance plots

+ call and put order imbalance are sum of 0DTE SPXW candle bid and ask volume every 5min, more selling (red), more buying (green), bins: {{ order_imbalance_bin_str }}.
+ in this doc we are defining "order imbalance" as sum of candle bid and ask volume, which is also referred to as "cumulative volume delta" - definately not true buy&sell. but maybe could be a poor man's guesstimate of order imbalance given a set duration.

# gamma exposure (gex)

+ gex is a guesstimate using 0DTE SPXW order imbalance (of entire day) from only the current day (akin @gexbot Classic, or @unusual_whales spot gex volume - not as accurate as @optionsdepth, @spotgamma Trace, @gexbot State)
+ horizontal dashed lines: spot: white, expected moves: gray, major positive gex strike: green, major negative gex strike: red.


# convexity

+ 0DTE SPXW order imbalance (of past 1hr) multiplied by current gamma, no put or call distinction. see gexbot document for more info, gexbot uses likely entire day order imbalance.
+ horizontal dashed lines: major positive convexity: cyan, major negative convexity: purple.
