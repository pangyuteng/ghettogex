# order imbalance plots

+ call and put order imbalance are sum of 0DTE SPXW candle bid and ask volume every 5min, more selling (red), more buying (green) {{ order_imbalance_range_str }}
# gex plot

+ gex is a guesstimate using 0DTE SPXW order imbalance akin @gexbot classic from only the current day (not as accurate as @optionsdepth @spotgamma)
(TODO: ideally i want to sum the order imbalance for the past few days to get a guesstimate of dealer directional open interest)

# convexity plot

+ 0DTE SPXW order imbalance of the entire day multiplied by current gamma, no put or call distinction. see gexbot document for more info.
+ horizontal dashed lines: major positive convexity: cyan, major nevative convexity: purple.
